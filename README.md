## 安装

```xml
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependency>
    <groupId>com.github.fishedee</groupId>
    <artifactId>spring-boot-starter-security-boost</artifactId>
    <version>1.0</version>
</dependency>
```

在项目的pom.xml加入以上配置即可
